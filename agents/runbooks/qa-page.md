# Runbook: QA a page (UI/perf/a11y)

## Checklist
- Responsive layouts across breakpoints
- Keyboard navigation works (tab order is logical)
- Focus states visible
- Headings: single H1, then H2/H3 in order
- Design consistency: use shared tokens/classes (no random spacing or custom colors), and reuse existing section components where possible
- Forms: labels + error messages
- Links: hover/focus states, correct targets
- Images: `alt` text and correct `sizes` when using `fill`

## Output
Produce a short list of fixes and apply them.
